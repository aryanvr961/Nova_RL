◈ REAL-WORLD PURPOSE ◈
🌍

Why This Benchmark Exists

🏦 DATA PIPELINES

Financial and enterprise ETL pipelines lose millions to silent data corruption. Nova RL trains agents that detect and repair these faults before they propagate to production.

🤖 RL RESEARCH

Provides a clean, deterministic environment with discrete action spaces — ideal for testing new RL algorithms like PPO, SAC, or GRPO on structured, non-game tasks.

🏥 HEALTHCARE DATA

Malformed records in clinical datasets directly impact patient safety. Nova RL's schema drift and type mismatch tasks mirror real hospital data quality challenges.

📦 SUPPLY CHAIN

Duplicate and null-heavy inventory data leads to stock mismanagement. Easy-tier tasks replicate exactly these high-frequency, high-cost warehouse data failures.

📡 ML DATA QUALITY

Poor training data quality degrades model performance. Agents trained on Nova RL serve as automated data gatekeepers before features enter the ML training pipeline.

🏆 LLM EVAL

Evaluates instruction-following LLMs on structured decision-making tasks, bridging the gap between text generation benchmarks and tool-use / agentic capabilities.

Action Space

  • fix — repair the anomaly directly
  • quarantine — isolate for review
  • promote — advance clean records
  • noop — skip (costs reward)
  • finalize — commit & end episode

Each action accepts: threshold · notes · params{}

◈ API ENDPOINTS ◈
🌐

Runtime API

GET /ping Health check — confirm service is alive
GET /health Detailed service status and config
GET /reset Create or reset a session → returns session_id
GET /state Read current observation (requires session_id)
POST /step Apply a remediation action → returns reward + next state
